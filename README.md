

## Maya 2024 批量重命名工具 (AssetRenamerUI)

这是一个专为 Autodesk Maya 编写的 Python 批量重命名图形化工具，基于 `PySide2` 和 Maya 内部 API (`maya.cmds`) 构建。它提供了一站式的资产命名规范化解决方案，涵盖了从基础编号、正则替换到 Maya 专属冗余清理的各类场景。

### 🌟 核心功能

* **① 批量重命名 (基础名 + 编号)**
* 支持为所选对象分配统一的基础名称，并自动追加自定义位数的流水编号（如 `Asset_01`, `Asset_02`）。
* 支持层级智能识别：顶层组节点保持基础名，组内子节点自动递增编号。


* **② 前后缀管理**
* 一键为多个对象添加统一的前缀（如 `M01_`）或后缀（如 `_HD`）。


* **③ 查找替换与正则预设**
* 支持常规文本替换与**正则表达式**高级替换。
* **内置大量一键规范化预设**：
* **左右镜像互换**：智能识别并互换命名中的 `L` ↔ `R` 或 `Lf` ↔ `Rt`。
* **Maya 残留清理**：一键去除粘贴复制残留（`pasted__`、`Copy_`）、默认节点前缀（`pCube1`、`polySurface`）以及 `Shape/Orig` 后缀。
* **格式规范化**：空格转下划线、去除多余合并下划线、去除非 ASCII 字符（中文/乱码过滤）。




* **④ 大小写转换**
* 支持一键将命名转换为全部大写、全部小写或首字母大写。


* **⑤ 精确字符裁剪**
* 自定义从命名的头部或尾部移除指定数量的字符。


* **⑥ 实用工具**
* **清理命名空间**：一键剔除导入资产自带的 `Namespace:` 前缀。
* **按类型加后缀**：自动检测节点类型（Mesh, Joint, Camera, Curve 等），并智能追加行业标准后缀（如 `_GEO`, `_JNT`, `_CAM`）。



### 🛠️ 技术亮点

* **UUID 底层防错机制**：在执行批量重命名时，脚本底层采用节点 UUID 进行追踪，有效避免了因父级节点被重命名而导致子级节点路径 (DAG Path) 失效、找不到对象的致命报错。同时，重命名操作采用由深至浅（倒序）的执行策略以确保层级安全。
* **智能节点过滤**：在遍历层级时，工具会自动识别并跳过所有 Shape 节点及隐藏的历史节点 (`intermediateObject`)，防止破坏物体的几何数据关联。
* **撤销栈支持 (Undo)**：所有的批量重命名操作均被封装在 `cmds.undoInfo(openChunk=True)` 中，用户可以通过 Maya 的 `Ctrl + Z` 一键撤销整个批量操作。

### 🚀 安装与使用

1. 将本 Python 脚本放入 Maya 脚本目录（如 `Documents/maya/2024/scripts/`），或直接在 Maya 的 Script Editor (Python 标签页) 中粘贴代码。
2. 运行以下代码启动界面：
```python
import maya.cmds as cmds
# 假设脚本名为 batch_renamer.py
import batch_renamer
batch_renamer.show_window()

```
