Maya 2024 批量重命名工具 (Batch Renamer)
这是一个基于 PySide2 编写的 Maya 2024 资产批量重命名脚本。它通过直观的 UI 界面，为建模师、绑定师和动画师提供了一站式的节点命名管理解决方案。该工具支持层级遍历，并底层通过 UUID 追踪节点，确保在复杂组层级下批量修改名称时路径依然准确安全。

✨ 核心功能 (Features)
① 批量重命名 (Batch Rename)

支持“基础名 + 自动补全零的编号”格式（如 Asset01）。

支持层级模式：开启后，用户选中的顶层组节点仅使用基础名，而其内部的子孙节点会自动按 Outliner 顺序递增编号。

② 前后缀管理 (Prefix & Suffix)

一键为选中的多个对象批量添加自定义前缀或后缀。

③ 强大的查找与替换 (Search & Replace)

支持基础文本替换以及正则表达式 (Regex) 替换。

内置大量常用自动化模板：

左右镜像互换：L ↔ R, Lf ↔ Rt 智能互换。

段落提取：按特定规则保留或删除首尾段落。

Maya 残留清理：自动去除默认前缀（如 pCube1）、Shape 冗余后缀（Orig/Deformed）、以及粘贴复制残留（pasted__/Copy_of）。

命名规范化：空格转下划线、合并连续下划线、去除非 ASCII 中文字符等。

④ 大小写转换 (Case Conversion)

支持一键切换为全部大写、全部小写或首字母大写 (Title Case)。

⑤ 字符裁剪 (Trim)

通过指定数值，精准移除名称首部或尾部特定数量的字符。

⑥ 实用拓展工具 (Utilities)

清理命名空间：一键抹除残留的命名空间冒号及前置字符 (namespace:name → name)。

按类型加后缀：自动识别节点类型（Mesh、Joint、Camera、Light 等），并智能补全标准行业后缀（如 _GEO, _JNT, _CAM, _LGT）。
